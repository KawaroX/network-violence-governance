网络暴力治理AI系统 API文档

本文档介绍了网络暴力治理AI系统提供的API端点及其使用方法。

基本信息

基础URL: /api/v1

所有API响应均为JSON格式，包含以下字段：

  • status: "success" 或 "error"
  • code: HTTP状态码
  • data: 响应数据 (成功时)
  • error: 错误信息 (失败时)

API端点

POST /api/v1/analyze

分析内容中的网络暴力倾向，提供微观内容分析和宏观话题分析。

请求参数

{

}

GET /api/v1/topics

        
获取所有话题列表。


        
响应示例

        
{
  "status": "success",
  "code": 200,
  "data": {
    "topics": [
      {
        "topic_id": "topic-12345678",
        "keywords": ["关键词1", "关键词2"],
        // 其他话题字段...
      },
      // 更多话题...
    ],
    "count": 1
  }
}
